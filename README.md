# To-Do List App ✅

A task management web app built using HTML, CSS and JavaScript.  
Made as part of my **CodeAlpha Web Development Internship** — May–August 2026 batch.

---

## 🔗 Live Preview
Open `index.html` in any browser — no installation needed.

---

## ✨ Features

- ➕ **Add** new tasks with a priority level
- ✅ **Mark tasks as complete** with a checkbox
- ✏️ **Edit** any task using the edit modal
- 🗑️ **Delete** tasks
- 🎯 **Priority levels** — Low 🟢 / Medium 🟡 / High 🔴
- 🔍 **Filter** tasks by All / Active / Completed
- 🧹 **Clear completed** tasks in one click
- 💾 **localStorage** — tasks are saved and persist after page refresh
- Works on mobile and desktop

## 🚀 How to Run

1. Download or clone this repository
2. Open the `index.html` file in your browser
3. Start adding tasks!

> Tasks are automatically saved to your browser's localStorage — they will still be there after you close and reopen the browser.
